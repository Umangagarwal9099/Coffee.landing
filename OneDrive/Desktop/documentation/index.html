Summary

This API reference includes technical documentation needed to formulate/standardize exchange API endpoints. Outlined below are the recommended and mandatory requirements. The public GET request endpoints are intended to allow access to market data.End points return results in JSON format.

Section A: Public API

  • /api/v1/public/markets - Overview of market data for all tickers and all markets.
  • /api/v1/public/tickers - Overview of ticker for all markets.
  • /api/v1/public/ticker - Overview of ticker for all markets.
  • /api/v1/public/book - Overview for all books.
  • /api/v1/public/history - Overview for all history.
  • /api/v1/public/history/result - Overview for all history.
  • /api/v1/public/products - Overview for product.
  • /api/v1/public/symbols - Symbols.
  • /api/v1/public/depth/result - Overall results.
  • /api/v1/public/kline - Overall Kline.
